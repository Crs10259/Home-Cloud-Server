Creating a New User

When you create a new user, they will have immediate access to the cloud storage system with the permissions and quota you set.

User Roles
  • Regular User: Can only manage their own files and view their profile.
  • Administrator: Has full access to all system features, including user management, settings, and system monitoring.
Storage Quota

The storage quota limits how much disk space a user can use. This can be changed later in the user edit page.
